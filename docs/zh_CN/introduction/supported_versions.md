|----------------|---------|------|
| 18.x | ✅ 完全支持 | 最新版本 |
| 17.x | ✅ 完全支持 | |
| 16.x | ✅ 完全支持 | |
| 15.x | ✅ 完全支持 | |
| 14.x | ✅ 完全支持 | 插件支持有保障 |
| 13.x | ✅ 完全支持 | |
| 12.x | ✅ 完全支持 | |
| 11.x | ✅ 完全支持 | |

## 功能可用性（按 PostgreSQL 版本）

- 简单查询
- 事务支持

| JSON 类型 | 9.2+ |
| JSONB 类型 | 9.4+ |
| UPSERT (ON CONFLICT) | 9.5+ |
| 并行查询 | 9.6+ |
| 存储生成列 | 12+ |
| JSON 路径查询 | 12+ |

## Python 版本支持

| Python 版本 | 支持状态 | 说明 |
|------------|---------|------|
| 3.14 | ✅ 完全支持 | |
| 3.14t (自由线程) | ❌ 不支持 | [psycopg issue #1095](https://github.com/psycopg/psycopg/issues/1095) |
| 3.13 | ✅ 完全支持 | |
| 3.13t (自由线程) | ❌ 不支持 | [psycopg issue #1095](https://github.com/psycopg/psycopg/issues/1095) |
| 3.12 | ✅ 完全支持 | |
| 3.11 | ✅ 完全支持 | |
| 3.10 | ✅ 完全支持 | |
| 3.9 | ✅ 完全支持 | |
| 3.8 | ✅ 完全支持 | |

**注意：** 自由线程版本的 Python 不受支持，因为 psycopg 尚未支持。进度请关注 [GitHub issue #1095](https://github.com/psycopg/psycopg/issues/1095)。
